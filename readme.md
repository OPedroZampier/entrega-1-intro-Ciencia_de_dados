# 📊 Análise Exploratória de Dados: Emendas e Transparência Pública

## 🎯 Visão Geral do Projeto
Em um cenário onde a transparência da informação é fundamental, este projeto tem como objetivo realizar uma análise de dados focada em emendas públicas. Mais do que uma simples manipulação de planilhas, esta é uma exploração prática de como os fundamentos da Ciência de Dados podem transformar dados brutos do mundo real em informações claras, estruturadas e estratégicas.

## 💡 Motivação e Contexto
Lidar com dados públicos geralmente envolve o grande desafio de organizar informações complexas e, muitas vezes, não padronizadas. O foco deste estudo foi aplicar o raciocínio analítico e a lógica para limpar, estruturar e extrair *insights* reais desse conjunto de dados. Afinal, por trás de cada linha e coluna, existe uma alocação de recursos e um fluxo de informações que precisa ser compreendido.

## 🛠️ Competências e Técnicas Aplicadas
Durante o desenvolvimento desta análise, foquei nas seguintes áreas:
- **Raciocínio Analítico e Lógica:** Estruturação do problema antes de ir para o código.
- **Tratamento e Estruturação de Dados:** Preparação da base para garantir resultados precisos.
- **Análise Exploratória (EDA):** Busca por padrões, anomalias e tendências dentro das emendas.
- **Comunicação e Storytelling:** Tradução de dados técnicos para conclusões claras e acessíveis.

## 🚀 Impacto e Aprendizados
Desenvolvido inicialmente como um projeto prático para a graduação em Ciência de Dados (UniCEUB), este repositório serve como um laboratório vivo das minhas habilidades. Ele reflete minha disciplina para o aprendizado contínuo, minha base em estatística aplicada e, principalmente, meu foco em usar a tecnologia e a comunicação para resolver problemas concretos.
